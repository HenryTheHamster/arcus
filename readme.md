# ensemblejs - skeleton project

```plain
  - game/
    - audio/
    - css/
    - js/
      - events/
      - gen/
      - levels/
      - logic/
      - modes/
    - scss/
    - views/
      - overlays/
      - pages/
      - partials/
```

## Run all your tests, build your css and browserified JavaScript.
```shell
gulp
```

## Start a local development server (in port 3000)
```shell
gulp local
```
